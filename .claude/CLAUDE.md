# mattbho.com blog

Personal brain-dump blog. Hugo + PaperMod, authored in Org mode via ox-hugo.

## Stack

- **Source:** `.org` files in `content-org/` - never edit `content/` by hand
- **Export:** ox-hugo converts Org → Hugo Markdown locally via `emacsclient`
- **Build:** Hugo + PaperMod theme (git submodule at `themes/PaperMod/`)
- **Deploy:** `git push` → GitHub Actions runs Hugo → GitHub Pages at mattbho.com

## Writing workflow

### Starting a post

Use `/new-post` with a title and a brain dump of what you want to say.
Claude will draft the post in your voice - rough, thinking-out-loud, no polish - then open it in Emacs.
Edit freely; you also have Claude available inside Emacs.

### Publishing

Use `/publish` from the blog repo directory.
It exports the current post via ox-hugo, commits, and pushes.
GitHub Actions deploys automatically - live within ~1 minute.

## File structure

```
content-org/          # SOURCE OF TRUTH - .org files only
  posts/
    my-post.org
  about.org
content/              # generated by ox-hugo - never edit directly
  posts/
    my-post.md
  about.md
themes/PaperMod/      # git submodule - never edit
static/CNAME          # mattbho.com
.github/workflows/
  deploy.yml          # builds with Hugo, deploys to Pages
```

## New post template

```org
#+hugo_base_dir: ../../
#+hugo_section: posts
#+title: Your Title Here
#+date: YYYY-MM-DD
#+hugo_draft: false

Content here.
```

## ox-hugo export command

```
emacsclient --eval '(progn (find-file "<abs-path-to-org-file>") (org-hugo-export-wim-to-md))'
```
